# 图片与媒体

## 功能列表

| 工具 | 功能描述 | 核心能力 |
|------|----------|----------|
| QR 码生成 | 多功能二维码生成 | 支持文本、URL、vCard、电话、邮件、短信、位置、事件、WiFi、支付；自定义颜色/Logo/容错等级 |
| QR 码解码 | 二维码识别 | 上传图片解码；调用摄像头实时扫描 |
| 图片转 Base64 | 图片编码 | 图片转 Base64 字符串；支持/不包含 data URL 前缀两种模式 |
| 图片压缩 | 图片压缩优化 | JPEG/PNG/WebP 压缩；质量调节；尺寸缩放 |
| 图片编辑 | 图片在线编辑 | 裁剪、旋转、翻转、镜像；亮度/对比度/饱和度滤镜 |
| 图片坐标 | 像素坐标拾取 | 点击图片获取坐标；支持百分比、千分比、万分比三种格式 |
| EXIF 查看 | 照片元数据提取 | 相机参数、GPS 定位、拍摄时间等 EXIF 信息 |
| 表情包分割 | 智能网格分割 | 自动检测网格结构并分割表情包图片 |
| 取色器 | 颜色值获取与转换 | HEX、RGB、HSL、HWB、LCH、CMYK、CSS 颜色名；实时预览 |

## 用户价值

- 社交媒体运营和设计师的日常图像处理工具
- QR 码生成器支持丰富的业务场景（WiFi 分享、电子名片等）
- 无需安装专业软件即可完成基础图像处理
