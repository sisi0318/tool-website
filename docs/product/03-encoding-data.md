# 编码与数据

## 功能列表

| 工具 | 功能描述 | 核心能力 |
|------|----------|----------|
| 编码解码 | 常见编码格式互转 | Base64 编解码、URL 编码解码、Hex 编解码；支持多行模式 |
| JSON 工具 | JSON 全方位处理 | 格式化/压缩/校验、JSONPath 查询、JSON-YAML 互转、Unicode-中文互转、转义/反转义、JSON 树形查看 |
| Protobuf 解析 | Protocol Buffer 数据处理 | 无 Schema 模式智能解析；基于 .proto 文件的解析和编码 |
| 进制转换 | 数字进制互转 | 二进制、八进制、十进制、十六进制、Base64 互转；支持自定义进制 |
| 大小写转换 | 文本大小写格式转换 | 全大写、全小写、标题大小写、camelCase、PascalCase、snake_case、kebab-case |

## 用户价值

- 开发者在 API 调试、数据传输时的必备编码工具
- JSON 工具集覆盖了日常 90% 的 JSON 操作需求
- Protobuf 解析器可在无 .proto 文件时快速查看二进制数据
