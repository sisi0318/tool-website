# 网络工具

## 功能列表

| 工具 | 功能描述 | 核心能力 |
|------|----------|----------|
| WHOIS 查询 | 域名 WHOIS 查询 | 基于 RDAP 协议；通过 IANA dns.json + RDAP 服务器查询；12h 缓存 |
| HTTP 请求调试 | HTTP 客户端 | 支持 Method/URL/Headers/Body/Params 配置；完整响应查看 |
| 设备信息 | 浏览器/系统环境检测 | 浏览器/OS/屏幕信息；网络 IP（通过 API）；电池状态；硬件信息；浏览器指纹（Canvas/WebGL/Audio） |

## 用户价值

- WHOIS 查询帮助域名选购和安全分析
- HTTP 请求调试器是 API 开发的轻量测试工具
- 设备信息工具可用于浏览器兼容性调试和隐私检测
