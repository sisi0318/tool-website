# 格式转换

## 功能列表

| 工具 | 功能描述 | 核心能力 |
|------|----------|----------|
| 温度转换 | 8 种温度单位互转 | Kelvin、Celsius、Fahrenheit、Rankine、Delisle、Newton、Reaumur、Romer |
| 汇率转换 | 实时汇率换算 | ExchangeRate-API 实时汇率；支持多种货币同时输入；转换历史记录 |
| Docker 转换 | Docker 编排格式互转 | `docker run` 命令与 `docker-compose.yml` 互转 |
| Crontab 生成 | Cron 表达式可视化生成 | 可视化构建器；表达式校验；常用预设；下次执行时间预测 |

## 用户价值

- 温度转换覆盖冷僻温度标度，满足科研和工程需求
- 汇率转换方便跨国购物和出差场景
- Docker 转换降低容器编排的学习成本
- Crontab 生成器免除记忆 cron 语法的困扰
