# 文本分析

## 功能列表

| 工具 | 功能描述 | 核心能力 |
|------|----------|----------|
| 文本统计 | 全方位文本分析 | 字符/单词/句子/段落统计；中英文混合分析；阅读时间估算；词频统计 |
| 正则测试 | 正则表达式测试器 | 实时正则匹配；flags 配置；常见示例模板；速查表 |
| 文本对比 | 文本差异比较 | 文本差异高亮对比 |

## 用户价值

- 文案写作者的效率工具，快速获取文本统计信息
- 开发者调试和验证正则表达式的必备工具
- 代码或文档版本对比的轻量方案
